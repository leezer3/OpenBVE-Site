The train.dat file format: Version History

■ Contents

  • 1. BVE1
  • 2. BVE2
  • 3. openBVE

■ 1. BVE1

The following BVE1 version strings are recognised:

▶ BVE1200000
BVE1210000
BVE1220000

■ 2. BVE2

The following BVE2 version strings are recognised:

▶ BVE2000000
OPENBVE

■ 3. OpenBVE

From openBVE 1.5.3.0 onwards, the OPENBVE identifier may be optionally followed by a version string.

The changes for each version are described below:

OPENBVE1530 - Added the EbHandleBehaviour parameter to the #HANDLE section.
OPENBVE1534 - The parameters in the #DELAY section may now optionally be made up of a comma-separated list, to define the delay on a per-notch basis.
OPENBVE1535 - Two parameters- LocoBrakeNotches and LocoBrakeType added to the #HANDLE section.
